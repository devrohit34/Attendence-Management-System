ATTENDANCE MANAGEMENT SYSTEM - WEB VERSION
===========================================

SETUP INSTRUCTIONS:
1. Create a new folder for the project
2. Save all the provided files in that folder
3. Open index.html in a web browser

FEATURES:
✅ User Authentication (Login/Logout)
✅ Dashboard with Statistics
✅ Mark Attendance with real-time updates
✅ User Management (Add/Edit/Delete Users)
✅ Reports Generation (Monthly/Yearly)
✅ Leave Management (Apply, Approve, Reject)
✅ Responsive Design for mobile devices

DEMO CREDENTIALS:
Username: admin
Password: admin123

TECHNOLOGY STACK:
- HTML5
- CSS3 (with Flexbox/Grid)
- JavaScript (ES6+)
- Font Awesome Icons

HOW TO USE:
1. Login with demo credentials
2. Navigate through sidebar menu
3. Mark attendance for users
4. Generate reports by selecting month/year
5. Manage leave applications
6. Add new users as needed

NOTE: This is a frontend-only demo. For production use, 
you would need to add backend functionality with PHP/Node.js 
and a database like MySQL.
